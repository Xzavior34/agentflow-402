# AgentMarket Contracts

## Quick Start

```bash
cd contracts
npm install
cp .env.example .env
# Edit .env with your PRIVATE_KEY
```

## Deploy to Cronos zkEVM Testnet

```bash
npm run deploy:zkevm
```

## Seed the Marketplace

```bash
npm run seed:market
```

## Run Live Simulation

```bash
npm run simulate
```

## Full Demo (Seed + Simulate)

```bash
npm run demo
```

Visit: https://cronos.org/faucet
